# Related work

## NoiseLight — environmental light & sound recordings

A companion sub-study used a custom in-house bedside sensor to record environmental light and noise levels during each patient's ICU stay. That work was published separately:

> Leone MJ, Dashti HS, Coughlin B, Tesh RA, Quadri SA, Bucklin AA, Adra N, Krishnamurthy PV, Ye EM, Hemmige A, Rajan S, Panneerselvam E, Higgins J, Ayub MA, Ganglberger W, Paixao L, Houle TT, Thompson BT, Johnson-Akeju O, Saxena R, Kimchi E, Cash SS, Thomas RJ, Westover MB. **Sound and light levels in intensive care units in a large urban hospital in the United States.** *Chronobiol Int.* 2023 Jun 3;40(6):759-768. doi: 10.1080/07420528.2023.2207647. PMID: 37144470; PMCID: PMC10524721.

Code repository (kept separate, not consolidated into this one): https://github.com/bdsp-core/noiselight
